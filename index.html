What is Marksmith?

Marksmith is a native macOS markdown editor. Open a file and edit it. Open a folder and see your markdown files in a clean tree view. Toggle between Edit and Preview with a single click. That's it.

No vaults. No subscriptions. No proprietary formats. Your files stay exactly where they are. Marksmith just helps you edit them.

Features

Full GitHub Flavored Markdown support: headings, bold, italic, tables, task lists, fenced code blocks, blockquotes, strikethrough, images, and links. Syntax-highlighted editing with a live preview. Folder mode with a sidebar tree view. Standard macOS keyboard shortcuts. Dark mode support.

Privacy

Marksmith has no network access. It does not collect, store, or transmit any data. Your files stay on your Mac.
